Задачей агента поиска всех стилей, используемых в html-документе является нахождение всех стилей, которые используются в html-документе. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей действию поиска всех стилей, используемых в html-документе. Аргументом запроса является html-документ, в рамках которого нужно найти стили. Результатом выполнения команды является конструкция, описывающая все стили, используемые в данном html-документе.
