Команда поиска всех стилей, используемых в html-документе предназначена для поиска всех стилей, которые используются в html-документе, поданном на вход данной команде. Аргуметом запроса является html-документ. Результатом выполнения команды является найденные стили, используемые в исходном html-документе.
