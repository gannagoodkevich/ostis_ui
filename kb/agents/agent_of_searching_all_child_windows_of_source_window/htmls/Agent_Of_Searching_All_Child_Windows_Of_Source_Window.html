Задачей агента поиска всех дочерних окон исходного окна является поиск всех окон, которые является дочерними для исходного. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска всех дочерних окон исходного окна. Первым аргументом запроса являются окно, дочерние окна которого необходимо найти. Результатом выполнения команды являются все найденные дочерние окна исходного окна.
