Задачей агента определения частоты ошибочного выполнения команд является вычисление количества безуспешных команд для пользователя. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу подсчета количества безуспешно выполненных команд. Единственным аргументом является пользователь, для которого высчитывается количество безуспешных команд.
