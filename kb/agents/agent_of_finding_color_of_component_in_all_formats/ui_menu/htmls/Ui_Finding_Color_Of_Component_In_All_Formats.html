Команда поиска цвета компонента во всех форматах предназначена для нахождения цвета указанного компонента во всех возможных форматах. Единcтвенным аргументом команды является компонент, имеющий цвет. Результатом выполнения команды является отображение цвета указанного компонента во всех возможных форматах.
