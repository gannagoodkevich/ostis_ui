Задачей агента поиска цвета компонента во всех форматах является нахождение цвета для для указанного компонента во всех форматах. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска цвета компонента во всех форматах. Единственным аргументом запроса является компонент, имеющий цвет представленный во всех форматах.
