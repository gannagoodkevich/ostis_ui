Команда поиска стилей конкретного вида, используемых в html-документе предназначена для поиска всех стилей, которые используются в html-документе, поданном на вход данной команде. Первым ргументом запроса является html-документ, в рамках которого нужно найти стили. Вторым аргументом является вид стиля. Результатом выполнения команды является найденные стили конкретного вида, используемые в исходном html-документе.
