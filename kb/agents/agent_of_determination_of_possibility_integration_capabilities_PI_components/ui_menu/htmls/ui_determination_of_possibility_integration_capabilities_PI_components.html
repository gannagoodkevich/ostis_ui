Команда определения возможности интеграции компонентов ПИ предназначена для определения возможности интеграции одного компонента в другой, поданных на вход данной команде. Первым аргументом запроса команды является компонент ПИ, в который осуществляется интеграция, а вторым - интегрируемый компонент ПИ. Результатом выполнения команды является вывод сообщения о возможности интеграции компонентов.
