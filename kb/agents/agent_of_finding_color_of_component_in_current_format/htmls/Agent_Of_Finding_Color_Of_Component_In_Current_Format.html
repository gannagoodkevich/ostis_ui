Задачей агента поиска цвета компонента в выбранном формате является нахождение цвета для для указанного компонента в заданном формате. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска цвета компонента в заданном формате. Аргументами запроса является компонент, имеющий цвет представленный в выбранном формате, и выбранный формат.
