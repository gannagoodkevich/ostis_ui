Команда поиска цвета компонента в выбранном формате предназначена для нахождения цвета указанного компонента в указанном формате. Аргументами команды являются компонент и формат, в котором необходимо вывести цвет компонента. Результатом выполнения команды является отображение цвета указанного компонента в указанном формате.
